# Cafe-Management-System
This is a beginner friendly Cafe Management System built with Python. It runs in the terminal and simulates a real cafe ordering experience from browsing the menu to getting a printed receipt.
A simple command-line cafe ordering system built using Python.  
This project allows users to view a menu, place orders, and receive a detailed receipt with total cost and timestamp.

## 🎯 Objectives

The main objectives of this project are:

- To build a basic real-world simulation of a cafe ordering system
- To practice Python programming fundamentals (loops, functions, dictionaries, and classes)
- To implement user input handling and validation
- To calculate total bill dynamically based on user orders
- To generate a formatted receipt with date and time
- To improve problem-solving and logical thinking skills in Python

## 🛠️ Features

- Display menu with item prices
- Take multiple item orders from the user
- Input validation for incorrect entries
- Quantity-based billing system
- Automatic total calculation
- Receipt generation with timestamp

## 💡 How It Works

1. The program displays the cafe menu
2. The user selects items and quantities
3. Orders are stored in a dictionary
4. Total bill is calculated
5. User can choose to print a receipt
6. Receipt shows all items, quantities, and final price
